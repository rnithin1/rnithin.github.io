Sim2Real for Self-Supervised Monocular Depth and Segmentation
Nithin Raghavan, Punarjay Chakravarty, Shubham Shrivastava
ArXiv Preprint, 2020

A twin VAE-based architecture with a shared latent space and auxiliary decoders is able to bridge the sim2real gap and produce real depth and segmentation maps without requiring any paired, ground-truth data in the real domain.
Fourier Features Let Networks Learn High Frequency Functions in Low Dimensional Domains
Matthew Tancik*, Pratul Srinivasan*, Ben Mildenhall*, Sara Fridovich-Keil, Nithin Raghavan, Utkarsh Singhal, Ravi Ramamoorthi, Jonathan T. Barron, Ren Ng
NeurIPS, 2020 (Spotlight)
project page / arXiv / code

Mapping input coordinates with simple Fourier features before passing them to a fully-connected network enables the network to learn much higher-frequency functions.
